# Contributing to Autodocs

We love contributions! Please follow these steps to contribute.

## Development Setup

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build all packages:
   ```bash
   npx turbo build
   ```

## Workflow

- Create a branch for your feature.
- Make changes.
- Add tests.
- Run `npx turbo test`.
- Submit a PR.

## Code Style

We use ESLint and Prettier. Run `npm run lint` and `npm run format`.
